# Калькулятор НДС

Простой одностраничный калькулятор для расчёта налогов (НДС + КПН) для Казахстана.

## Ссылка

- Сайт: https://me-qr.kz/nds/
- Репозиторий: https://github.com/mr-diamant/nds-calculator

## Как работает

1. Пользователь вводит "чистую сумму" — сумму, которую хочет получить на руки.
2. Калькулятор рассчитывает сумму с налогом, умножая чистую сумму на коэффициент `1.312217`.
3. От полученной суммы с налогом вычисляются:
   - НДС — 16%
   - КПН — 10%

## Формулы

### Режим по умолчанию — КПН от суммы с налогом

```
total = clean / (1 - 0.16 - 0.10)
nds   = total × 0.16
kpn   = total × 0.10
```

### Режим "КПН за минусом налогов"

```
total = clean / (1 - 0.16 - 0.10 × (1 - 0.16))
nds   = total × 0.16
kpn   = (total - nds) × 0.10
```

Переключатель находится на странице под полем ввода суммы.

## Файлы

- `index.html` — весь сайт (HTML + CSS + JS).

## Локальный запуск

Просто откройте файл `index.html` в браузере. Для разработки удобно использовать любой локальный сервер, например:

```bash
python3 -m http.server 8000
```

После этого откройте http://localhost:8000.
